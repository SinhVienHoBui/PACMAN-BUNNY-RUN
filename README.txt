Project Name: Pacman-Bunny Run

The following instructions will guide you through setting up and running the code.

Prerequisites:
- A modern web browser (e.g., Chrome, Firefox, Safari)
- A text editor or IDE (e.g., VSCode)
- A local web server (e.g., Live Server extension for VSCode, Python's SimpleHTTPServer)

Setup Instructions:
1. Download or clone the project repository to your local machine.
   git clone https://github.com/SinhVienHoBui/PACMAN-BUNNY-RUN.git

2. Open 'menu.html' with Live Server or 'index.html' to nevigate directly to the main game screen.
